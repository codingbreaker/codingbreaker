## About

I'm a developer and automation engineer from India. I spend most of my time writing Python scripts that automate things I'd otherwise have to do manually — and then automating those scripts too.

My work sits at the intersection of **software development**, **AI tooling**, and **process automation**. I've built CLI utilities, web scrapers, email outreach pipelines, AI-powered workflows, and 3D web experiences. I care about shipping real things that actually work.

I use Claude Code daily as part of my development workflow — I also built a [token count statusline](https://github.com/codingbreaker/claude-code-count-statusline) for it that runs on every platform.

```
Location   →  India
Role       →  Developer & Automation Engineer
Focus      →  Python · AI workflows · CLI tools · Web
Status     →  Open to freelance and collaboration
Portfolio  →  codingbreaker.github.io/portfolio.github.io
```

`Python` `CLI` `Cross-platform`

```
Features:
  · Live token usage display in terminal statusline
  · Cross-platform: Windows, macOS, Linux
  · Zero dependencies beyond Python stdlib
  · Integrates directly with Claude Code session
```

My actual development environment — nothing exotic, just what works.

```yaml

Node     : 20 LTS
```
